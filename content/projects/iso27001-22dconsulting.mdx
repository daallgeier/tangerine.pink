Throughout 2022 I led the implementation of ISO27001 for 22d consulting. This involved:

- Creating a risk register
- Creating a risk treatment plan
- Creating a Statement of Applicability
- Creating a set of policies
- Creating a set of procedures
- Creating a set of work instructions
- Creating a set of forms
- Creating a set of records

and more. 

I also created a set of training materials and delivered training to the team.  Our certification audit was successful and we were awarded ISO27001 certification in November of 2022.
